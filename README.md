# Gul Bil
et lille spil med hjemmeside og arduinoer som input
## hjemmeside:
- opret bruger/login
- lave nyt spil, vise score, vælge spillere/arduino
- vi laver det så der kommer biler på skærmen med forskellige farver
## api
- håntere opstart af spil, gemme og håndter score
- user management, avartar (S3 minio)
- snakke med rabbitMQ
## Arduino
- controller til spillet
- uniqe id
- info om spillet på skærmen
- vise connection status
- (mulig udvidelse: stød armbånd)
